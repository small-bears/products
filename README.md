#### 三、 项目介绍
基于SpringBoot的高校请假管理系统包含教师、管理员、学生三个模块，具体功能如下：

1. 教师模块（教师使用）
功能覆盖：系统首页、学生管理、公告信息、请假记录、考勤记录、个人中心
2. 管理员模块（系统管理角色使用）
拥有全局管理权限，功能包括：系统首页、学生管理、教师管理、请假类型管理、公告信息、请假记录、考勤记录、系统日志、个人中心
3. 学生模块（学生使用）
功能聚焦请假及个人事务：系统首页、公告信息、请假记录、考勤记录、个人中心

#### 四、项目技术

是否前后端分离项目：是

后端： SpringBoot+Mybatis-Plus

前端：Vue + Vue Router + ELementUI + Axios

#### 五、 运行环境

开发语言：Java

项目架构：B/S架构

开发工具：IDEA，Eclipse,Myeclipse都可以。推荐IDEA

JDK版本：1.8

数据库: MySQL8.0版本以上

项目管理：Maven

Node版本：14



#### 六、安装教程

使用Navicat或者其它工具，在mysql中创建对应名称的数据库，并导入项目的sql文件；

使用IDEA/Eclipse导入项目，若为maven项目请选择maven;导入成功后请执行maven clean;maven install命令，然后运行；

进入src/main/resources修改application.yml里面的数据库配置

启动项目
